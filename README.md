# Amoherom VTuber Tools

A collection of personal tools I built to make editing and animating **VRM models** in **Unity** faster, easier, and more fun!  
Originally, I was doing all the animations and edits manually, but I decided to turn my workflow into tools so I can improve without stressing out too much.  
I’ll keep adding new tools as I develop them, so this will grow over time.  

---

## 🚀 Features

- Latest: Thumbnail Camera
- Planned: Collison Ragdoll Generator and editor (mostly for cloth physics) 

---

## 🛠 Installation

1. Fork or clone the repository
2. Open the project in **Unity **.
3. Start using the tools in your workflow!

or,
   
1. Once the Release is uploaded, Download the latest Release,
2. Import it as Custom Package
3. Start Using it on your project

*Important : [UniVRM](https://github.com/vrm-c/UniVRM) Required!. 

*Unity Version im Using: 2022.3.62f2 (Others sometimes gave me a MToons shader issue so im using this)
